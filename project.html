Website Project

When this project started, our first work was completing the Code Academy courses and get the proper and enough information about creating a basic website with HTML,CSS, and JavaScript. After finishing the courses, Ibrahim Asgarov started to work on website, and created the skeleton of the website. Basically, he created the pages of the website, gave them a design with CSS, and set a form with JavaScript which is "Contact Us". After him, Kanan Hajiyev started to fill the inside of the website with the needed information and media. Adjusted the hypertexts and checked the website if there is any bug or not. Then Urfan Zeynalov created the logo of the website and helped with the design. We faced with lots of problems like bugs or dead hypertexts but fortunately with the help of the course we corrected them and gained practical skills through that.
